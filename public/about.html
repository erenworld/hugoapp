À propos

Ce site vous permet d'écrire et de stocker des notes sur votre appareil dans un environnement sans distraction.

Les notes sont sauvegardées automatiquement dans votre ordinateur, de sorte qu'elles sont privées et ne sont accessibles que par vous. Vous pouvez télécharger une seule note sous forme de fichier texte, ou exporter la base de données entière pour la restaurer ou la transférer sur un autre ordinateur et/ou navigateur.

Scrivi.la est également une application Web progressive : vous pouvez l'installer et l'exécuter hors ligne.

Nihal.app est gratuit : pas de scripts d'analyse, pas de publicité, juste IndexedDB pour stocker les notes. Utilise quelques icônes Phosphor. Si vous trouvez ce site utile, n'hésitez pas à m'offrir un café. Et n'hésitez pas à le partager.
